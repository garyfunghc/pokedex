## [Demo](https://garyfunghc.github.io/pokedex/)

## Tech stack
- NextJS
- Typescript
- Axios
- [PokeAPI](https://pokeapi.co/) Restful API
## Starter
```
yarn build
yarn start
```
## Basic features
- List Pokemons
- Show Pokemon detail
- Add favourite Pokemon and store in localstorage
- Search Pokemon by name
